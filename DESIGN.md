# Diseño

## Tesis visual
App doméstica limpia, útil y cálida. Debe sentirse como una herramienta real, no como una demo genérica SaaS.

## Paleta
- Fondo: azul petróleo oscuro
- Superficie: azul grisáceo
- Acento: verde lima para comprado
- Alerta: coral suave
- Texto: casi blanco

## Estilo
- mobile-first
- tarjetas compactas
- contraste alto
- botones grandes
- esquinas redondeadas

## Reglas
- la acción principal siempre visible
- el estado comprado debe verse instantáneamente
- el nombre de quien compró debe ser explícito
- máximo 1 nivel de navegación

## Anti-patrones
- tablas densas
- demasiados modales
- gradients startup genéricos
- iconografía excesiva
