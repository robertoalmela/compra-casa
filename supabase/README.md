# Siguiente paso real

Para convertir este MVP en app compartida real Android+iPhone gratis:

1. Crear proyecto Supabase free.
2. Ejecutar `supabase/schema.sql` en SQL editor.
3. Añadir `supabase-js` al frontend.
4. Reemplazar localStorage por lectura/escritura realtime contra `shopping_items`.
5. Crear flujo simple de hogar + código de invitación.
6. Publicar en Vercel/Netlify gratis.
7. Añadir al móvil como app desde navegador.

## Objetivo MVP real
- un hogar
- varios miembros
- lista compartida en tiempo real
- item comprado + quién lo compró
- sin App Store / Play Store

## No hace falta
- app nativa
- pagos
- backend propio
- infraestructura compleja
